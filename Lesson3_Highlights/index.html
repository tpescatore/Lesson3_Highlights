Highlight #1 Git

Git is a virtual control system. It is an online server that is used to run small to large projects. It runs them with high speed and it is very efficient. All of the data on git is stored locally and the user can work on it offline. It was made in 2005.

Highlight #2 Version Control Systems

Version Control Systems allow users to track file changes. They are very useful for writers, designers, and developers. The types are local, centralized, and distrubuted version control systems.

Highlight #3 Git Commants

Git has many commands. An example "git init" initializes or creates a file; "git status" will show you what files have been modified and ready to be added; "git add" will allow you to add new content for committment; "git commit" will allow you to finalize putting content up; "git push" will allow you to push everything up to the remote repository.

Highlight #4 Local Repository

A local repository is the Repository that resides on a local machine of an individual users. Some ways to creat it is to create a brand new repository. Another way is to clone an existing remote repository onto your local machine

Highlight #5 Remote Repository

A remote repository is a repository that resides on a remote server that is shared among multiple team members. It is the place where the code is stored.
